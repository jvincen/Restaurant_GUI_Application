# Online Food Ordering Website

This project is an online food ordering website implemented using Java, JavaFX, Eclipse IDE, and Scene Builder. It provides users with a platform to browse food items, add them to a shopping cart, and place orders.

## Key Features

1. **CRUD Operations on Food Items**: Implemented functionalities to enable users to perform CRUD (Create, Read, Update, Delete) operations on food items within the shopping cart, enhancing user experience and flexibility in managing their orders.

2. **Order List Feature**: Designed and implemented an Order List feature that provides customers with:
   - Unique Order ID: Each order is assigned a unique identifier for tracking purposes.
   - Real-time Insights on Queue Status: Users can view the status of their order in the queue, providing transparency and reducing uncertainty.
   - Estimated Order Preparation Time: Provided customers with an estimated time for order preparation, improving user satisfaction and expectation management.

## Project Structure

The project is structured as follows:

- `Main.java`: Entry point of the application responsible for initializing the GUI and starting the main functionality.
- `Login.java`: Class handling the login functionality for users.
- `Card.java`: Class managing the addition of payment cards for users.
- *Other Java files*: Additional classes and components contributing to the functionality of the application.
